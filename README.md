# Growing_city_Unity

### 지속적으로 확장 가능한 2D 방치형 게임</br></br>

#### 특징:</br>
+ 메뉴 4개 존재, 2개는 초당 골드 획득량을 증가, 2개는 초당 골드를 획득해줌.
+ 초당 돈의 액수를 더더욱 효과적으로 생성해주는 다른 맵 존재.
+ 일일 상점, 캐시 상점 및 재접속 보상 존재.</br></br>

#### 쓰여진 기법:</br>
+ 구글 플레이 연동하기 위해 JSON 직렬화.
+ 구글 플레이 인앱 결제 시스템 연동.
+ 유니티 배너 및 동영상 광고 존재.
+ 상속을 통해서 지속적으로 확장 가능한 구조.
